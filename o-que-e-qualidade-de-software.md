## **O que é Qualidade de Software?**

A qualidade de software garantem que um sistema atende aos requisitos esperados, funciona corretamente e proporciona uma boa experiência ao usuário. Ou seja, um software de qualidade é aquele que faz o que deveria fazer, sem erros, com eficiência e segurança.


### *Objetivo da Qualidade de Software:*
* Funcione corretamente
* Atenda aos requisitos do cliente
* Seja seguro
* Tenha bom desempenho
* Seja fácil de usar
* Tenha poucos ou nenhum bug

##

### A qualidade de software não é só “não ter erro”. Ela envolve vários fatores:


*Funcionalidade:*
* O sistema faz o que foi solicitado?

*Confiabilidade:*
* Ele funciona sem falhar?

 *Usabilidade:*
* É fácil de usar?

*Performance:*
* É rápido e eficiente?

*Segurança:*
* Protege os dados do usuário?

*Manutenibilidade:*
* É fácil de corrigir e evoluir?


##

### *Exemplo de qualidade na prática::*

Imagine um sistema de login:

1. Aceita senha errada -> problema de funcionalidade

2. Demora 10 segundos -> problema de performance

3. Interface confusa -> problema de usabilidade

4. Dados vazam -> problema de segurança

#### *Tudo isso impacta a qualidade do software.*

##
O profissional de QA é **responsável por garantir a qualidade do sistema.**

Ele atua criando cenários de teste, encontrando bugs, validando funcionalidades, garantindo que requisitos foram atendidos, prevenindo problemas antes da produção

QA não é só testar, é garantir qualidade em todo o processo.
